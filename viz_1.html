Purpose

The purpose of this prototype is to visualize number of alarms in different Neighborhoods of San Francisco, based on Priority.

Wrangling

For this prototype, reduce dataset to 5 years and removed null values from Neighborhoods, as well as Priority. Sorted graph as per number of alarms.

Tableau Prototype

Prototype was done using Tableau.

Theme Analysis

Prototype 1

So my prototype tells you about number of alarms in different area of San Francisco. It shows maximum number of calls from Tenderloin and most of these calls are prioritized as Emergency. Many of these calls are for life threatening incidents. Least calls are received from Presidio and Treasure Island.

Prototype 2

So this prototype shows different alarms encoded over color and prioritized over size, where 3 is Emergency. It shows Station Area over different zip code. It shows that 94110 has highest station area but least Priority. It relates with first prototype which shows different Neighborhood of SF.

Prototype 3

So heatmap shows that Tenderloin has highest calls and highest response time. Most of the calls are "Life threatening". It shows top 10 Neighborhoods of San Francisco.cd
